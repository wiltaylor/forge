## Default Permission

Full Forge IPC surface: contract requests, events, and widget sessions.

#### This default permission set includes the following:

- `allow-request`
- `allow-widget-open`
- `allow-widget-send-text`
- `allow-widget-send-binary`
- `allow-widget-close`
