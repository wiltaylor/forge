# forge

Home of the **Forge design system** and its tooling, split out of the `meta` repo.

- `.claude/skills/forge-design/` — the Forge design system skill: dark-default,
  dense SolidJS components (shell, forms, feedback, data, charts, node graph,
  code editor), design tokens, and a runnable preview gallery
  (`preview/` — `npm run dev`).
- `.claude/skills/playpen/` — the playpen skill: builds server-backed
  interactive playground apps (FastAPI + SolidJS, Forge-styled) in a
  `.playpen/` folder.

Both live under `.claude/skills/` so they're active for Claude Code sessions
in this repo.
